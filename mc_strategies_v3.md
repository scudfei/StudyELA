## 一、最重要的前提认知

### 1.1 两类题目共存，规则不同 ◆

原文在全部四本 KEY 书末均有明确表述：

**普通题（有唯一正确答案）：**
> "Each question has one correct answer. The other answers are distractors, which are incorrect."

**评估题（最佳答案题）：**
> "Some questions may ask you to select the best answer. These questions will always include words like **best, most appropriate, most likely, most strongly, most clearly, most accurately, most completely**. All of the answers will be correct to some degree, but one of the choices will be better than the others in some way."

**实操意义：** 先看题目里有没有以下词语——

```
best · most appropriate · most likely · most strongly
most clearly · most accurately · most completely
```

有 → 所有选项都部分正确，找"最支持"的那个  
没有 → 找唯一正确的那个，其余三个是错的

**关于 main / mainly / primary / primarily：◇（新增，原文无此触发词清单）**

`main` / `mainly` / `primary` / `primarily` 不在上面官方列出的 best-answer 触发词中，因此不要自动把它们当成"所有选项都部分正确"的评估题。

更稳妥的处理方式：

```
先判断题目问的是哪一种"主要"：
□ 全文主旨 / controlling idea → 选覆盖全文最多的选项
□ 段落或局部功能 → 选最贴近该局部证据的选项
□ 主导手法 / main device → 选出现密度最高、组织全文结构的手法
□ main reason → 选题干限定行内最直接的因果理由

只有当两个以上选项都明显有文本支撑时，才进入 Section 5 的比较。
```

这条是为了防止把 `main` 误用成 `best / most clearly`。`main` 的核心不是"四项都对"，而是"哪个最符合题干限定的范围"。

**思维模式重构：○（作者推断，但有"best answer"逻辑支撑）**

对理科生来说，ELA 选择题的目标不是"找到完美答案"，而是"排除三个被文本推翻概率最大的选项"。剩下那一个即为正解。这与实验中排除无效假设的逻辑一致。

### 1.2 原文关于"文本理解"的核心提示 ◆

> "Consider everything in relation to the **controlling idea**."  
> "The multiple choice questions also require **close reading**."  
> "It is just as important to understand the **question** as it is to understand the text."

这三句话的实操含义：
- 答题前先确定全文的核心观点（controlling idea）
- 每道题都要回到文本中精读相关段落，不凭记忆
- 认真读题干——题目问的是什么，决定了什么选项是正确层级的

## 二、原材料官方推荐策略（完整版）◆

以下内容逐字来自 KEY Strategies 章节，是 Alberta Education 官方建议：

1. **快速浏览全卷** — 先了解题目总数，合理分配时间
2. **先想答案，再看选项** — "Try to think of an answer before looking at the choices"（防止被选项引导）
3. **关键词下划线** — 在题干和选项中圈出关键词
4. **图形题处理顺序** — 看图 → 读题 → 返回图，圈出题目中的重要信息
5. **排除明显错误** — "Eliminate those choices that are clearly wrong or do not make sense"
6. **评估题读完全部四项** — 在最佳答案题中，必须读完所有四个选项再做选择
7. **标注难题，跳过后返回** — 在难题旁标"*"，继续作答，最后返回
8. **后题可提供线索** — "Sometimes clues are given throughout the test that will provide you with answers"（后面的题有时能帮助回答前面的题）
9. **注意否定词** — "Note any negative words like **no** or **not** and be sure your choice fits the question"
10. **不要轻易改答案** — "Before changing an answer, be sure you have very good reason"
11. **不要找规律** — "Do not look for patterns on your answer sheet"

## 三、题型专项策略

### 3.1 词义题："In the context of lines X–Y, [word] means..."

**这类题的核心陷阱，原文用具体例子明确指出：◆**

> "Students often do poorly when the surface meaning of text is not the real meaning."

原文举例（10-1 KEY，第719行）：

```
In the context of lines 5–8, anxious means
A. eager     B. worried     C. doubtful     D. uncertain
```

答案是 **A（eager）**，不是 B（worried）。

原文解释：
> "You might choose worried because anxious does mean worried. But the question refers to lines 5–8... 'Premier Jones is anxious to read the results...' Worried to read? That cannot be right. Worried is not used with the preposition to. The only word that fits is eager."

**操作步骤：◆（步骤来自原文示范）**

```
第一步：不看选项，读原文段落
第二步：判断这个词在这个句子中的实际功能
        （它修饰什么？它后面跟的是什么搭配？）
第三步：把每个选项代入原句，测试语法和语义是否通顺
第四步：选代入后整句读起来最自然的那个
```

**五类上下文线索（原文明确列出五类，不是四类）◆：**

| 线索类型 | 识别方式 | 示例标志词 |
|---------|---------|---------|
| 定义（Definition） | 句子直接给出解释 | is, refers to, means |
| 举例（Example） | 举具体例子说明 | for example, for instance |
| 重述（Restatement） | 换一种说法重复 | or, which is, in other words |
| 对比（Contrast） | 通过反义来暗示 | but, however, unlike, while |
| 比较（Comparison） | 通过相似性暗示 | like, similar to, as if |

### 3.1b 当两个选项意思相近时：微观修饰语溯源 ◇

当词义题或语气题中两个选项都"感觉对"，普通阅读层无法区分时，用以下方法：

**操作：回到文本，提取该段落中所有修饰名词和动词的词（形容词、副词）作为数据集，再与选项匹配。◇**

```
示例 A：数据集 = faded / golden / childhood / warm / long ago
→ 这组词指向：Nostalgic（怀旧），而非 Melancholic（忧郁）

示例 B：数据集 = grey / heavy / sinking / cold / silent
→ 这组词指向：Melancholic（忧郁），而非 Wistful（惆怅）
```

原文要求"阅读时标注显著词汇（significant words）"◆，此技巧是对该要求的操作化。
两组修饰词所指向的情感质地不同，这差异就是区分相近选项的证据。

### 3.2 语气/情感/态度题

**Tone 和 Mood 的精确定义（原文，三本 KEY 均一致）◆：**

> **Tone** = "The writer's apparent attitude toward his or her **subject**, characters, or readers."  
> **Mood** = "The atmosphere of a piece of writing **that is felt by the reader**."

注意：原文从未将 mood 定义为"角色感受的氛围"，而是"读者感受到的氛围"。两者的主体不同：

| 概念 | 属于谁 | 问题判断 |
|------|-------|---------|
| **Tone** | 作者 | "作者对这个主题/人物的态度是什么？" |
| **Mood** | 读者 | "读者读这段文字时会感受到什么氛围？" |

**操作建议：◇（原文有 tone/mood 定义，此步骤是作者延伸）**

先判断题目问的是 tone 还是 mood，再从文本中找具体词汇支撑。
- 找描述性词汇、情感词、修饰语
- 题目中有 "author" 关键词 → 问 tone
- 题目中有 "reader feels" / "atmosphere" → 问 mood

**关于"连续谱"排列语气词：○（作者推断，原文无此框架）**  
原文仅列举了若干 tone 词的例子（serious, formal, sympathetic, disgusted 等），并未提供任何光谱或量化排列方式。

### 3.2b 结尾氛围题：三轴法 ◇

当题目问 `atmosphere / mood / concludes in an atmosphere of`，不要只抓最后一个情感词。Mood 是读者感受到的整体氛围，尤其要看结尾之前的冲突是否仍在。

**操作：把氛围拆成三轴，每轴只做低成本判断。**

| 轴 | 判断问题 | 常见证据 |
|---|---|---|
| Tension | 紧张是否仍存在？ | silence, argument, threat, unresolved disagreement, guarded body language |
| Resolution | 问题是否真正解决？ | explicit agreement, final decision, accepted plan, restored relationship |
| Polarity | 最后情绪偏正、偏负还是混合？ | relieved, intrigued, fearful, exhausted, joyful, uncertain |

**重要区分：◇**

```
relieved = 压力暂时减轻，不等于 optimistic
intrigued = 有兴趣/被吸引，不等于 convinced 或 resolved
contented resolution = 需要明确满足 + 问题解决
strained uncertainty = 可以同时容纳"暂时松一口气"和"根本问题未解决"
```

学生实操版本：

```
□ 冲突还在吗？
□ 有人明确同意/问题明确解决了吗？
□ 最后一个正向词是不是只是"松一口气"，而不是"乐观期待"？

如果答案是：冲突还在 + 没明确解决 + 只有暂时缓和
→ 优先选能容纳 tension 和 uncertainty 的选项；置信度最多中。
```

### 3.4 文学手法识别题

这类题最接近"知识题"，考查的是对术语定义的精确掌握。以下区分均来自原材料定义：◆

**最容易混淆的对组：**

| 易混淆组 | 区分依据（来自原文定义） |
|---------|----------------------|
| **simile vs metaphor** | 有 like / as 的比较 = simile；直接断言 A is B = metaphor |
| **verbal irony vs sarcasm** | 两者都是"字面意思与实际意思相反"；但 sarcasm 带有讽刺攻击性（cutting tone）◆ |
| **situational irony vs dramatic irony** | situational = 结果与预期相反；dramatic = 观众知道但角色不知道◆ |
| **allusion vs symbol** | allusion = 指向文本外的参照物（圣经/历史/文学）；symbol = 文本内的具体物代表抽象概念◆ |
| **tone vs mood** | tone = 作者的态度；mood = 读者感受的氛围◆ |

**原文特别警示（10-1 KEY 原文）◆：**

> "Students often do poorly when the surface meaning of text is not the real meaning."

反讽和讽刺题是失分重灾区。对理科生来说，需要特别建立一个**"反向雷达"开关**：

```
触发条件（任一满足即开启）：
□ 文本中出现夸张（Hyperbole）：如"我饿到能吃下一头大象"
□ 文本语气与内容严重不匹配：冷静地讨论极端事物（如 Swift 的 A Modest Proposal：
   平静地"建议"把爱尔兰婴儿作为食物——这篇文章攻击的是英国地主阶级的压迫，
   而非真的提供食谱）
□ 句子中出现引号包裹的词（表示"所谓的……"）

开启后的操作规则：◇
→ 最符合字面意思的选项，优先排除
→ 提供象征性/批判性解释的选项，优先保留
```

（此操作规则是对原文警示的推导，原文仅指出失分模式，未提供具体排除步骤）

### 3.4a 修辞手法题：四轴证据公式 △

修辞手法题最容易错在"文中确实有这个手法，但它不是题干问的那个答案"。例如同一段文字可能同时有反讽、夸张和讽刺目的。v3 的核心不是问"哪个手法存在"，而是问：

```
正确答案强度 =
题干范围匹配
+ 题干对象匹配
+ 术语必要特征匹配
+ 题干效果桥接强度
- 强干扰项解释力
```

#### 第一步：范围轴 Scope Gate △

先圈出题干限定的范围。范围决定选项层级。

| 题干范围词 | 优先检查 | 常见误选 |
|---|---|---|
| phrase / word / image / sentence | 局部语言手法 | 把全文主题或作者目的误选成局部 technique |
| anecdote / example / comparison | 该例子内部最直接的手法和效果 | 只选全文存在的 broad device |
| paragraph / stanza / scene | 局部功能：推进、对比、转折、强调 | 只回答一个句子细节 |
| passage / essay / poem / overall | 全文主导目的或组织手法 | 只抓一个局部亮点 |
| statement / view / assumption / claim | 观点本身的逻辑形态：paradox、irony、fallacy 等 | 把观点涉及的故事当成 allegory |

学生实操版本：

```
题干问哪里，就只先在那个范围里找证据。
如果一个选项只在全文层面成立，但题干问某个局部，它要降权。
如果一个选项只解释一个词，但题干问 overall effect，它也要降权。
```

#### 第二步：对象轴 Object Gate △

同一段文字可以同时有 technique、tone、purpose、theme。先判断题干问哪一种对象。

| 题干问法 | 需要回答的对象 | 选项应是什么 |
|---|---|---|
| What technique/device is used...? | 写法手段 | irony, hyperbole, metaphor, allusion 等 |
| What tone/attitude is conveyed...? | 作者/说话者态度 | mocking, critical, sympathetic 等 |
| What is the purpose/effect...? | 为什么这样写 / 造成什么效果 | criticize, emphasize, contrast, reveal 等 |
| What idea/theme is suggested...? | 文本想表达的观点 | 关于人、社会、权力、关系的抽象判断 |
| What image is created...? | 读者看到/感到的人物或事物形象 | evil, vulnerable, absurd, dignified 等 |

若题干问 `technique`，不要直接选"批判社会"这类目的；若题干问 `purpose`，不要只停在"用了比喻"这类手段。

#### 第三步：术语必要特征卡 △

以下是考场用的最小判定条件。它们不是完整文学理论定义，而是选择题排除干扰项时的必要特征。

| 术语 | 必要特征 | 反向排除 |
|---|---|---|
| **Hyperbole** | 明显夸大到超出现实可能或正常程度 | 只是强烈批评但没有放大，不够 |
| **Irony** | 表面意思、表面期待或表面结果，与真实意思/实际结果相反 | 只有负面态度，不一定是 irony |
| **Verbal irony** | 说出来的字面意思与真正意思相反 | 没有"反说"，可能只是批评或夸张 |
| **Situational irony** | 事情结果与合理期待相反 | 不是某句话"说反了" |
| **Dramatic irony** | 读者/观众知道，角色不知道 | 不是普通误会；必须有读者-角色信息差 |
| **Sarcasm** | 带攻击性或嘲弄性的 verbal irony | 没有刺伤对象时，只能算 general irony |
| **Satire** | 用幽默、反讽、夸张或荒诞批判社会、制度或普遍人性 | 只攻击一个局部人物行为，可能太窄 |
| **Paradox** | 看似矛盾，但能表达深层真理 | 真正逻辑错误是 inconsistency，不是 paradox |
| **Inconsistency** | 前后说法或逻辑无法同时成立，且没有深层真理 | 如果矛盾是有意揭示真理，优先 paradox |
| **Allegory** | 整个故事/人物/事件系统性映射另一套抽象意义 | 单个典故或单个象征不够 |
| **Allusion** | 指向文本外的宗教、历史、神话、文学、文化知识 | 文本内物象代表抽象概念是 symbol |
| **Symbol** | 文本内具体物反复承载抽象意义 | 只出现一次且无暗示，证据弱 |
| **Motif** | 反复出现的意象、词语、情境或结构，累积主题意义 | 单次出现通常不够 |
| **Synecdoche** | 用部分代整体，或用整体代部分 | 只是相关联替代时，可能是 metonymy |
| **Metonymy** | 用与对象密切相关的事物代指对象 | 若是部分/整体关系，优先检查 synecdoche |
| **Analogy** | 用 A 与 B 的结构相似来解释观点 | 只是说 A 是 B，可能是 metaphor |
| **Juxtaposition** | 把两个不同事物放在一起，制造对照或意义张力 | 仅有差异但没有被并排呈现，证据弱 |
| **Contrast** | 强调两个对象的差异 | 不一定要求相邻摆放 |
| **Metaphor** | 直接把 A 说成 B | 有 like / as 的显性比较通常是 simile |
| **Simile** | 用 like / as 等标记进行比较 | 有 like / as 但不是比较时不能机械选 |
| **Personification** | 给非人事物人类动作、情感、意志或语言 | 普通动作动词不一定是拟人 |
| **Understatement** | 故意把严重/强烈的事说得很轻 | 与 hyperbole 相反，但也可能制造 irony |
| **Oxymoron** | 两个相反词构成短语 | 整句话的矛盾通常更像 paradox |
| **Ambiguity** | 文本有意保留多重可能意义 | 普通表达不清不一定是文学 ambiguity |
| **Incongruity** | 风格、情境、行为或期待之间不协调，常制造讽刺、荒诞或批判效果 | 普通差异不一定是不协调 |
| **Rationalization** | 人物为不合理行为、信念或选择找貌似合理的借口 | 普通解释原因不等于 rationalization |
| **Pathos** | 激起读者情感反应的诉求或效果，常由重复、意象、排比等制造 | 更常是效果，不一定是具体 device |
| **Rhetorical question** | 问句不是真的求答案，而是强调观点或施压 | 真正询问信息则不是 |
| **Repetition** | 重复词、句式、意象以强调或形成节奏 | 重复必须有可解释效果 |
| **Anaphora** | 连续句子或分句开头重复同一词语/短语 | 不是所有 repetition 都是 anaphora |
| **Parallelism** | 相似语法结构并列，制造节奏、平衡或强化 | 只是内容重复，不一定是 parallelism |

三份校验资料（Ultimate Guide / Knowledge Tree / Exam Competency Map）明确支持或重点列出的方向包括：metaphor、simile、personification、allegory、synecdoche、metonymy、symbolism、motif、allusion、oxymoron、paradox、hyperbole、understatement、verbal/situational/dramatic irony、sarcasm、satire、ambiguity、incongruity、rationalization、repetition、anaphora、parallelism、rhetorical question、pathos。

`Analogy`、`Juxtaposition`、`Inconsistency` 在上述三份校验资料中不是显性核心条目。若选项中出现，可以按通用术语逻辑检查；若选项中没有出现，不应把它们当作优先考点。

#### 第四步：效果桥接测试 △

把题干和选项连成一句话，比较哪个选项解释题干效果时跳步最少。

```
[题干效果] is created by [选项] because [文本可见特征].
```

或：

```
作者使用这个局部手法，是为了/从而 [题干问的效果]。
```

判断标准：

```
□ 能否指出文本里的可见特征？
□ 这个特征是否满足术语必要条件？
□ 这个术语是否直接解释题干关键词？
□ 是否需要绕到全文目的、背景知识或个人感觉才能说通？
```

若两个选项都说得通，优先选择"解释路径更短、题干关键词覆盖更直接"的选项。

#### 第五步：强干扰项排序 △

修辞题常见强干扰不是完全错误，而是"层级不精确"。

| 强干扰类型 | 表现 | 处理 |
|---|---|---|
| 手法存在但不是题干效果 | 文中确实有 irony，但题干问某种人物形象；另一个手法可能更直接制造该形象 | 降权，不立即删除 |
| 全文目的冒充局部手法 | 全文是 satire，但局部问某个 phrase 的 technique | 局部题优先局部手法 |
| 术语大类压过小类 | satire 可能包含 irony/hyperbole，但题干问具体 technique | 选更具体且可见的手法 |
| 效果正确但手法不准 | 选项说出了作者在批评，但不是题干问的 device | 按对象轴排除 |
| 术语必要条件缺失 | 想选 allegory，但没有整套系统映射 | 排除或低置信 |

#### 置信度规则 △

```
高置信：一个选项满足范围、对象、必要特征、效果桥接；其他选项缺少必要条件。
中高置信：两个选项都在文中存在，但其中一个更直接解释题干关键词。
中置信：需要依赖较多术语知识或文本外知识，且强干扰也有证据。
低置信：策略只能排除明显错误，剩余选项必须靠未掌握的术语定义区分。
```

此公式不能替代术语知识。它的作用是把术语知识转化为证据检查顺序，避免只凭"这个词我见过"或"这段感觉像"来选。

### 3.4b 文学映射题：至少两个锚点 ◇

有些题会把一个物象、动作、舞台安排或景物描写映射到人物性格、主题或关系上。例如植物、天气、道具、座位安排、服装、动作等。

这类题不能只凭"感觉像"。为了避免过度解读，使用**双锚点门槛**：

```
若要说 A 物象映射 B 人物/主题，至少找两个文本锚点：
□ 共享词汇：同一组词同时用于物象和人物
□ 共享动作：物象行为与人物行为结构相似
□ 共享冲突：物象处境与人物处境相同
□ 叙事邻近：作者紧接着把两者放在同一段或同一转折中
```

只有一个锚点时，保留但降置信度；没有锚点时，按类型 E（真实但无关）排除。

这条不是要求学生写长篇分析，而是防止"我觉得这个象征某某"的跳跃推断。

### 3.5 推断题（Inference Questions）

**原文对 inference 的定义 ◆：**

> "To make an inference, you need to **draw a conclusion about the meaning of text using all the given information**."  
> "Your background knowledge and the information in the text **create meaning beyond what is directly stated**."

原文的自问策略：◆
> *"What is the writer actually saying? What is the writer leading me to believe?"*

**关于"推断步数"的判断：○（作者推断，原文无此表述）**  
"一步推断合法，两步以上是陷阱"这条规则不来自原文，是作者归纳的启发式规则。实际上原文要求的是推断必须"基于文本中给定的全部信息"，超出这个范围的结论就不可靠。

### 3.5b 背景知识使用闸门 ◇

原文允许使用 background knowledge，但背景知识只能帮助解释文本给出的信息，不能替文本创造新证据。

**三步闸门：**

```
第一步：先找文本/图像锚点
        例：时代、地点、道具、人物身份、注脚、题目说明

第二步：问背景知识是否只是解释这个锚点
        可以：1885 London + monocle → 绅士时髦/社会身份
        不可以：monocle → 这个人一定傲慢/邪恶（文本无支撑）

第三步：若正确选项必须依赖背景知识才成立
        → 置信度最多中高；若文本锚点薄弱，标中或存疑
```

学生实操版本：

```
背景知识只能回答"这个文本细节通常意味着什么？"
不能回答"我知道这类人通常是什么样的人，所以他一定怎样。"
```

### 3.5c "most strongly" 的根因覆盖测试 ◇

当题干含 `impact / influence / effect on behaviour / reveals most strongly` 时，两个选项常见区别是：

| 选项类型 | 特征 | 权重 |
|---|---|---|
| 根因 / 全局约束 | 能解释多个后续行为、习惯、恐惧、选择 | 通常更强 |
| 局部表现 / 单次行为 | 只解释一个动作或一个情节结果 | 通常较弱 |

**操作：对两个候选做覆盖测试。**

```
候选 A 能解释几个后续行为？
候选 B 能解释几个后续行为？

若 A 能解释 3 个以上行为，而 B 只解释 1 个局部动作，
并且题干问的是 influence/impact most strongly，
→ A 优先。
```

**限制条件，防止过度套用：**

```
如果题干问 "directly shown by the action" / "specific behaviour" / "immediate result"，
则局部行为证据可能优先。

如果题干问 "impact / influence / effect"，
先检查是否有更高层的根因证据。
```

这条的目标不是永远选抽象选项，而是选**能解释更多文本行为的因果源头**。

### 3.6 叙事情节结构题：先排序，再判功能 △

> **适用范围说明：** 本节策略主要针对**叙事散文和短篇小说**。诗歌、议论文、说明文中是否存在同类题目，以及本框架是否适用，尚未经 Alberta 实题系统验证，遇到此类文本时谨慎套用。

这类题考的不是某个事件"重不重要"，而是这个事件在叙事结构中的功能。常见题干包括：

```
climax / turning point / rising action / conflict / complication
resolution / denouement / aftermath / ending functions to...
sets the conflict in motion / increases suspense / foreshadows
```

注意：`foreshadowing / conflict / ending function` 只能部分使用本策略。它们还需要结合对应题型判断：
- foreshadowing：先找后文实际发生的事件，再回头找前文暗示
- conflict：先确定冲突双方或冲突类型
- ending function：可能问 theme / tone / irony / characterization，不一定问 plot phase

**可信度说明：△ + ◆**

- `climax / rising action / resolution` 等术语定义属于通用文学和考试原则，并可由 Class Focus 的 plot elements 内容支持；但不是 KEY Strategies 原文直接策略。△
- "先排序，再判功能"是作者整理的考试操作法；具体执行仍依赖 KEY 原文已有原则：理解题干、close reading、回到文本精读相关段落。◆

**核心区分：**

| 阶段 | 判断问题 | 常见位置/功能 |
|---|---|---|
| Trigger / inciting incident | 是什么把问题启动了？ | 冲突开始或被点燃 |
| Rising action / complication | 风险是否继续上升？ | 问题扩大、压力增加 |
| Crisis | 是否出现重大危险或关键选择？ | 高风险事件，但未必是最高点 |
| Climax | 中央冲突是否达到最高张力、决定性变化，或结果开始无法回避的节点？ | 情节峰值、最大压力、决定性转折 |
| Falling action | 高潮后还有什么直接后果？ | 张力开始下降 |
| Resolution / denouement | 问题怎样收束？ | 结局、人物回到某种新状态 |
| Aftermath / thematic ending | 结尾展示什么反应或主题？ | 强化主题，但通常发生在 climax 之后 |

**操作步骤：**

```
第一步：圈出题干问的具体结构词
        是 climax？turning point？rising action？resolution？还是 ending function？

第二步：把四个选项按故事实际发生顺序排列
        不先判断哪个"看起来最重要"，先判断它们发生的先后。
        若文本有 flashback / frame narrative / non-linear order，
        按故事实际发生顺序，不按页面出现顺序。

第三步：确定 central conflict
        这个故事/诗的主要压力、危险、选择或问题是什么？

第四步：问每个事件之后张力是否还继续上升
        如果之后还有更高风险、更大损失、更关键选择或更决定性的结果，
        那么当前事件通常不是 climax，而是 trigger / rising action / crisis。

第五步：区分 climax 和主题性结尾
        结尾若只是展示他人反应、社会冷漠、人物回到日常，
        它可能强化 theme，但不一定是 climax。

第六步：选择最符合题干结构词的事件功能
```

**高频误判：**

```
误判 1：把 first cause 当 climax
        触发事故/冲突的事件通常是 trigger，不是最高张力点。

误判 2：把最显眼、最剧烈的事故当 climax
        事故可能只是 crisis；若之后风险、选择或损失继续升级，
        climax 可能在后面。

误判 3：把主题最明显的结尾当 climax
        结尾反应可能揭示主题，但如果发生在决定性结果之后，
        通常是 aftermath / denouement。

误判 4：只看单个事件，不看事件链
        情节结构题必须看 sequence：before → during → after。
```

**学生实操版本：**

```
看到 climax 题，不问"哪个事件最重要？"
改问：
□ 哪个事件先启动问题？
□ 哪个事件让风险继续变大？
□ 哪个事件之后已经没有更高的情节风险？
□ 哪个事件是最高张力、决定性变化，或结果开始无法回避的节点？
□ 哪个事件只是高潮之后的反应？
□ 不自动选死亡、失败或最后事件；先问它是否真的是 central conflict 的最高张力。
```

**例型，不记答案，只记结构：**

```
若选项依次是：
A 触发事件
B 重大危机
C 关键结果/决定性变化/最大张力点
D 其他人事后反应

题干问 climax：
→ 通常优先检查 C 是否真的是 central conflict 的最高张力点。
  如果 C 只是结局说明或事后反应，不选。

题干问 theme / ending effect：
→ D 可能更重要，因为它展示事后反应或主题。
```

**限制条件：**

```
turning point 不一定永远等于 climax。
有些文本的 turning point 是人物作出决定的时刻；
有些文本的 climax 是决定带来的最终冲突峰值。
crisis 和 climax 在有些文本中可能重合；
判断依据不是标签，而是该事件之后 central conflict 是否仍继续升级。
所以必须按题干具体词判断，不要把所有结构题都自动选"死亡/结局"。
```

**可审计输出格式：**

```
题干结构词：
选项故事顺序：
central conflict：
每个选项之后张力是否继续上升：
最终选择：
排除理由：
```

## 四、干扰项分析（明确标注为作者归纳）

**重要说明：以下五种干扰项类型不来自原始材料。**  
原文对干扰项的唯一说明是："distractors are incorrect"和"eliminate those choices that are clearly wrong"。以下分类是作者对考试题目规律的归纳，供参考，不保证完整或正确。◯

---

**类型 A：半对半错（Boolean AND Trap）**

```
形式化表述：Part₁(真) AND Part₂(假) = 整个选项(假)
只要复合选项中一个子句无文本支撑，整个选项出局。

操作：在连接词（and / but / which implies / showing that）处把选项劈开，
      分别在文本中验证每一半。
```

**类型 B：绝对词陷阱（Absolute Language Trap）**

```
结构：使用 always / never / entirely / completely / all / must / only / proves 等绝对词
风险：文学极少能"证明"某事，绝对化的选项容易是陷阱
应对：看到绝对词，在文本中找一个反例；找到即排除
```

反向参考——以下词语出现在选项中时，该选项**相对安全**，优先保留考虑：○
```
often · implies · suggests · may · primarily · tends to · reinforces
```
（注：这组"安全词"是基于标准化考试命题规律的推断，非原材料表述）

**类型 C：字典义陷阱（Dictionary Meaning Trap）◇**

```
结构：给出这个词最常见的字典义，但不符合文中的具体语境
风险：这是原文明确指出的失分模式（"anxious"例题）
应对：把选项代回原句，测试语法搭配和语义是否通顺
```

注：这条陷阱有原文例题支撑，可信度高于其他三条。

**类型 D：层级错位陷阱（Wrong Level Trap）**

```
结构：题目问"整体主题"，选项回答的是"一个具体细节"；或反之
风险：细节本身可能是真的，但不在正确的分析层级上
应对：先判断题目问的是"整体"还是"局部"，再过滤选项
```

**类型 E：真实但无关陷阱（True But Irrelevant Trap）○**

这种干扰项最难识别：**选项陈述的是文本中确实存在的客观事实，但它不能回答题干问的"为什么"或"效果是什么"。**

```
识别方法——"Because/To 桥接测试"：
把 [题干的动作或现象] + because/to + [选项] 连成一句话，
测试这个理由在逻辑上是否自洽。

示例：
题干：The use of the word "Pomeranian" primarily serves to...
选项：Show that the owner is wealthy.
桥接：作者用这个词，是为了展示主人很富有。
→ 逻辑牵强（博美犬并不必然代表财富），排除。

正确方向：Emphasize the irritating nature of the situation.
桥接：作者用这个词（博美以神经质叫声著称），是为了强调情境令人烦躁。
→ 逻辑自洽，保留。
```

（注：此桥接测试方法是将形式逻辑借用到文本分析的跨学科工具，不来自原材料）

## 五、当两个选项无法区分时的决策参考

**◯ 以下六个问题全部来自作者推断，原文无此框架，仅供参考：**

```
Q1：哪个选项需要依赖文本以外的知识才能成立？
    → 需要"文本外知识"的那个，通常是干扰项

Q2：两个选项的表述范围是否不同（一宽一窄）？
    → 如果题目用了"best describes/most clearly"：选更具体的
    → 如果题目问"主题"：选更概括的

Q3：两个选项中，哪个有否定词或绝对词？
    → 带否定词的选项需要格外核查文本

Q4：哪个选项与你在答题前（看选项之前）自己想到的答案最接近？
    → 原文建议先想再看，第一直觉通常更可靠
       "Before changing an answer, be sure you have very good reason to do so." ◆

Q5：哪个选项解释了更多文本证据？
    → 尤其用于 most strongly / influence / impact 题
    → 能解释多个行为的根因，通常强于只解释一个行为的局部表现

Q6：哪个选项把局部情绪误当成整体氛围？
    → 尤其用于 atmosphere / mood 题
    → 单个 relieved / interested / amused 不足以推出整体 optimistic / resolved
    → 若冲突未解决，选择能容纳残留 tension 的选项
```

## 六、速查卡（考场使用）

### 6.1 答题流程

```
【读题前】
□ 看题干中有没有 best / most / likely → 决定是"唯一正确"还是"最佳"题型
□ 看到 main / mainly / primary → 先判断范围：全文、局部、主导手法、主要原因
□ 在脑中先预判答案（不看选项）◆

【读选项】
□ 排除明显错误或读不通的 ◆
□ 代入词义选项测试语法搭配 ◆
□ 对评估题：读完全部四项再选 ◆
□ 对 most strongly / influence / impact：比较根因覆盖范围 vs 局部表现
□ 对 atmosphere：检查 tension / resolution / polarity 三轴
□ 对 climax / plot structure：先排故事实际顺序，再判断 trigger / rising action / climax / aftermath
□ 对修辞手法题：先锁范围和对象，再检查术语必要特征，最后做效果桥接

【无法确定时】
□ 回到文本，找相关段落精读 ◆
□ 注意题干中有没有 no / not ◆
□ 若仍无把握，猜一个，标"*"，继续 ◆
□ 全卷结束后返回，后面的题可能提供线索 ◆
```

### 6.2 核心概念对照

| 问题方向 | 答题要抓住的关键 |
|---------|--------------|
| 词义题 | 语境 > 字典义；代入测试语法是否通顺 |
| 语气题 | tone = 作者态度；mood = 读者感受；先定正负向 |
| 结尾氛围题 | tension / resolution / polarity 三轴；局部 relieved 不等于 optimism |
| 目的题 | 四类：inform / persuade / entertain / describe |
| 反讽题 | 表面意思 ≠ 真实意思；引号 / 夸张是信号 |
| 修辞手法题 | 范围 + 对象 + 必要特征 + 效果桥接；不是问"文中有没有"，而是问"哪个最直接解释题干效果" |
| 夸张 vs 反讽 | hyperbole 看是否极端放大；irony 看表面与真实是否相反；若两者都存在，看题干问形象、语气还是目的 |
| 悖论 vs 不一致 | paradox = 看似矛盾但揭示真理；inconsistency = 逻辑上不能同时成立且没有深层意义 |
| satire vs irony/hyperbole | satire 常是全文批判目的；irony/hyperbole 常是实现 satire 的局部手法 |
| symbol vs motif | symbol 是具体物承载抽象意义；motif 是反复出现的意象/词语/情境累积意义 |
| ambiguity / incongruity / rationalization | ambiguity 看多重意义；incongruity 看不协调；rationalization 看人物为不合理行为找借口 |
| repetition / anaphora / parallelism | repetition 是重复；anaphora 是句首重复；parallelism 是相似语法结构并列 |
| 推断题 | 必须能从文本直接引出；无法找到原文支撑的结论要怀疑 |
| 背景知识题 | 先找文本锚点；背景知识只能解释锚点，不能创造证据 |
| 文学映射题 | 至少两个锚点：共享词汇/动作/冲突/叙事邻近 |
| 情节结构题 | 先排故事实际顺序；climax = 中央冲突最高张力、决定性变化，或结果开始无法回避的节点；结尾反应可能只是 aftermath |
| 最佳答案题 | 所有选项都"对"，找文本支撑最强的那个 |
| most strongly 影响题 | 根因若能解释多个行为，通常强于单个局部表现 |
